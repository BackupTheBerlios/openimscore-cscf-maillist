Hi Min,

The OpenEPC project is not open source. Of course, as some parts are 
open source (e.g. the cdp stack), they will stay so and also get the 
updates and improvements that we'll be building as part of the OpenEPC 
project. Also it is quite probable that some modules which could be 
beneficial for the Open IMS Core for example will be opened in the near 
future.

However, we are not able to finance the extensive work required for 
OpenEPC from a 3rd party, as to offer this in the same manner that we 
did with the Open IMS Core. So the main part of the OpenEPC will remain 
closed source. We are trying though to price it so that it is affordable 
in most situations, with big discounts for academia for example. The 
first official release is planned around the end of March and you can 
find more details and prices from the website and from info at openepc.net .

Cheers,
-Dragos
