http://lists.berlios.de/pipermail/openimscore-cscf/2008-August/001881.html

This is because of an incompatability issue between the 08 cdp branch 
and the trunk code. The P-CSCF in the branch does not support the 
qop-parameter.  To solve the problem disable the qop parameter in the 
SCSCF config file, or take a look at the relevant code changes (see 
related thread) to incorporate into the branch code.
