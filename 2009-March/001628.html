I have just commited (r627) the changes of the IPSec client scripts,

have made them backward compatible, so things should not break if some
parameters are missing (fingers crossed).

TS 33.203. It is now the IPSec scripts that convert the value into
something usable by the 'setkey' command (tun -> tunnel, trans ->
