Dear RR,

In the P-CSCF module the parameters ipv4_for_signalling and 
ipv6_for_signalling are used for the Rx interface towards a PCRF (Policy 
and Charging Rule Function).

I am sorry if the name of the parameter miss lead you.

If you want to have 2 different network card, you can setup 2 P-CSCFs, 
each for every IP. OpenIMSCore might support using more than 1 IP for 
listening (

listen=109.239.50.67
listen=109.239.50.68

)
, but I will have to test.
