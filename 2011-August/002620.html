I think you have to dig deeper into the concept of TLS. The main steps are:
- you have to become a CA (Certificate Authority)
- create a CA  root certificate (self signed)
- create a server certificate for P-CSCF
- provide the signed server certificate to P-CSCF
- add the(self signed)  CA root certificate to the list of trusted CAs of
your client.

The details can be found in some tutorials on the web.
