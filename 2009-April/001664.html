If you run the CSCF on different machines and merge the pcap files you 
will have each packet duplicated, because one time sending and one time 
receiving.
The resend timer of SIP is per default 500 ms and not  5 µs. So I 
believe this is the same packet captured on different interfaces!

Also see packet 24 25, it is the same.

So you captured on 172.20.110.230 and 172.20.110.236 and merged these 
capture files. And so each packet which was sent between these two hosts 
are duplicated!
