Hi,

They are for S-CSCF persistency. The state of S-CSCF is periodically dumped
to DB (or files) and loaded at startup. Later this stuff will be included on
the scripts.

If you want to use them now, you have to create a db, create the tables that
are in one of those files and modify the scscf.cfg:

-         loadmodule "……/postgres.so or mysql.so"

-         modparam("scscf","scscf_db_url","postgres://user:pwd@localhost
/scscfdb")

-         set persistency_mode to 2 or 3.
