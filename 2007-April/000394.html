Hi Leigh,

According to 24.228 the Proxy-CSCF address can be obtained from the
PDP Context Activation signalling (when using GPRS access). This is
called "P-CSCF discovery procedure".
The UE can send SIP signaling towards the core only after knowing the
P-CSCF address (its "outbound proxy"), so only after establishing the
PDP context.

JF
