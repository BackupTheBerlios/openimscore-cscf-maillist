Hello heiko,

you can't now trigger that directly from the script. There was a
discussion some time ago if anybody was interested in writing the Sh
wrapper module. This modules would need to implement the Sh commands
while using the cdp module, like the scscf/icscf modules do now for the
Cx interface (maybe also extracting the Cx interface in a separate
module might help?).

As I said before, I started drafting such a module, but unless somebody
would be interested in contributing in building it, I wouldn't add it as
it is just a draft. If you are interested, we would happily open a
branch for that and I would commit the start-up thing there.

-Dragos
