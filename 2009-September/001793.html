OpenIMSCore wrote:
> Some clients (Nokia-Siemens demo clients distributed to their partners) do
> not include an Expires header when registering.
>
> Unfortunately, OpenIMSCore interprets an absent expires header as 0
> (unregister) rather than 3600 as recommended by RFC 3856, so these clients
> can never register.
>
> The patch attached below addresses the issue.
