Hi Ramachandran,

24.229 7.6.0, in chapter 5.4.1.2.2    Protected REGISTER  - step 11) a)
- indicates that the Path should be copied from the REGISTER to the 200 OK.

11) create a 200 (OK) response for the REGISTER request, including:

a) the list of received Path headers;


So I contradict you. Anyway, the Path is copied on purpose there and
does not get copied by default.
