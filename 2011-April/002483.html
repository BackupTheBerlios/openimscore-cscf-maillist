The call of the scripts is embedded into the code of P-CSCF. Follow e.g. the
function call " P_security_401" in pcscf.cfg. The function is implemented in
"security.c" module of pcscf.
There you will see how all parameters are prepared to finally call the
script with " execute_cmd(cmd)".
