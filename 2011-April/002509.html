Hi all,

Is there any way to define trusted entity for OpenIMS ? What we did is 
that we modified code and added configurable to scscf.cfg so based on 
URI OpenIMS is able to recognize trusted entity within the network.

typical scenario :

subs A (invite) -> IMS -> AS(application server) -> IMS -> subs B

In case AS is not defined in OpenIMS as a trusted entity INVITE going 
out from AS won't be processed by IMS since AS is not authenticated.
