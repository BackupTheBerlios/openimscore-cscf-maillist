> No, thats not the way it is defined.
> The Service-Route header field (only) has to be used by the
> UE to route initial and standalone requests by using a
> proper Route header.
> The Path header field is only for the S-CSCF which stores
> the values to route terminating requests properly. The UE
> has to ignore the Path header field.
