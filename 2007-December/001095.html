Hi,

You can use Early-IMS and most probably NBA (I haven't tested the latter yet...).
Therefore you need to enable EarlyIMS and disable the prefered authentication method (or select HSS-selected) in scscf.cfg

#modparam("scscf","registration_default_algorithm","AKAv1-MD5")
#modparam("scscf","registration_default_algorithm","AKAv2-MD5")
#modparam("scscf","registration_default_algorithm","MD5")
#modparam("scscf","registration_default_algorithm","CableLabs-Digest")
#modparam("scscf","registration_default_algorithm","TISPAN-HTTP_DIGEST_MD5")
# Let the HSS decide
modparam("scscf","registration_default_algorithm","HSS-Selected")

modparam("scscf","registration_disable_early_ims",0)

Then provision the IP-address of your client to the EarlyIMS field on IMPI (Public User ID) level...for my tests I've also enabled only Early-IMS as authentication scheme and Early-IMS-Security as default authentication method, but actually this should not be necessary...

br, Klemens
