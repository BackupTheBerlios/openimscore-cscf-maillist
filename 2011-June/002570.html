Hi all,
Just reporting back how I resolved this issue. The reason for this behavior was that my UE was using a different port for ipsec protected messages then the port for unprotected messages; as soon as I started using the same port for both protected and unprotected; the issue with SUBSCRIBE was resolved.
best regards,
