Hi,

I am currently working on the debian package for the OpenIMSCore project.
I follow the howto for debian (from
http://lists.berlios.de/pipermail/openimscore-users/2008-July/005840.html )

# Prearation
1. sudo apt-get install dpkg-dev
2. cd /usr/local/src
# Checkout ser_ims
3. svn co http://svn.berlios.de/svnroot/repos/openimscore/ser_ims/trunk ser_ims
4. cd ser_ims
5. make deb
6. cd ..
# Install default ser packages
7. sudo dpkg -i ser_0.9.4-0.2-svn573_i386.deb
ser-mysql-module_0.9.4-0.2-svn573_i386.deb
ser-presence-module_0.9.4-0.2-svn573_i386.deb
ser-xmlrpc-module_0.9.4-0.2-svn573_i386.deb
# Remove ser service
8. /etc/init.d/ser stop
9. update-rc.d ser remove
# Install Open IMS core pcscf, icscf, scscf
10. dpkg -i ser-scscf_0.9.4-0.2-svn573_i386.deb
ser-pcscf_0.9.4-0.2-svn573_i386.deb ser-icscf_0.9.4-0.2-svn573_i386.deb
ser-cscf_0.9.4-0.2-svn573_i386.deb

At point 5, the make deb command result is good but I do not have any .deb
package in the parent directory.
Is there a new way to make the package ?
