Currently, the only encryption/security mechanism used on the Gm
interface (P-CSCF-UE) is the 3GPP recommended IPSec.  owever, this is
not spread too much in the clients, which usually implement just TLS.
This feature is set at enabling TLS on the P-CSCF and investigating if
it would be possible to use the keying material provided by AKA.
