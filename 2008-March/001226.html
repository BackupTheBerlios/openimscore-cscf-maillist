Hi Victor

from your question in the mailing list i guess you only want to add this 
parameter for the ISC interface to the AS.

the files  and functions you are interested in are  under  modules/isc/

specially file mark.c  and file mod.c

in mark.c you find the functions that really do the work (i think easy 
to understand and well documented by Dragos)
and in mod.c the functions you call from the configuration script

You can either change them to add the parameter (notice that orig/term 
are given as a parameter of the functions in mod.c  
ISC_match_filter("orig") in the cfg file is going to be str1 in the 
mod.c function) or follow the way this is done here to create your own 
ones....

if you need further help just post your question to the list
