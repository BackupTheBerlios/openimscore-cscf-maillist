Hi Dragos,

thanks to Mark Evans from Sprint we have solved the char conversion issue at
the debug-log presentation of the IP-address in module registration.c of
scscf.

Actually the line 576 in registration.c reads:
   LOG(L_ERR,"DBG:"M_NAME":S_is_authorized: IP address in MAA was
<%d.%d.%d.%d>\n",

This produces an output string of 
   DBG:S-CSCF:S_is_authorized: IP address in MAA was <-64.-88.0.3>

Please change line 576 to:
   LOG(L_ERR,"DBG:"M_NAME":S_is_authorized: IP address in MAA was
<%hhu.%hhu.%hhu.%hhu>\n",

This produces now the correct output string of:
   DBG:S-CSCF:S_is_authorized: IP address in MAA was <192.168.0.3>


Maybe there are other places where this conversion might be helpful.

Cheers
Franz
