To help you analyse the problem I need two things:

1. the trace-file should not be a textfile but a file in pcap-format
produced inside the Linux machine with "tcpdump -i any -s 3000 -w
/var/tmp/tracefile.pcap".

2. Add a trace-statement into pcscf.cfg and scscf.cfg to mark a request when
it arrives. I do this with the command at the beginning of themain routing
block:
     xlog("L_INFO","%CybTracepoint: %cs %rm %ru from %si:%sp From: %fu  To:
%tu\n%Cxx");
    Therfore you also have to load the xlog module: loadmodule
"/opt/OpenIMSCore/ser_ims/modules/xlog/xlog.so"
