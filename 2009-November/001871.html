Hi,

I'm using eclipse with the CDT plugin, but mainly because it's a 
cross-language platform and it easily support also the Java stuff quite 
well. Also the editor is quite helpful with code refactoring, visual 
version diff, doxygen tasks a.s.o.

Otherwise, I would recommend that the building and svn control would be 
done manually, from a console. Although eclipse can help, I am using it 
here just as a graphical help, but not as the core tool.

Cheers,
-Dragos
