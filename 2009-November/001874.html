> the situation is as follows:
>
> Within a network you can define capability sets (see in HSS GUI 
> “Network Configuration”).
>
> Capability sets consist of different capabilities which can be given a 
> name and which can be set to mandatory or optional.
>
> These capability sets can be assigned
>
> - to the subscriber (see HSS GUI “IMS Subscription”), and
>
> - to S-CSCF (see I-CSCF tables “scscf” and “scscf-capabilities”). 
> There is obviously no GUI to set these values.
>
> During initial registration (see TS 24.229 §5.3.1.2) the I-CSCF 
> selects the S-CSCF which fits best for the subscriber based on the 
> capability data received in UAA.
>
> /The ICSCF … select a S-CSCF that fulfils the indicated mandatory 
> capabilities – if more than one S-CSCFs fulfils the indicated 
> mandatory capabilities the S-CSCF which fulfils most of the possibly 
> additionally indicated optional capabilities;/
