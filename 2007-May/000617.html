Hello Abhishek,

You have unfortunately posted on the wrong mailing list as the CSCF do
not have anything to do with the respective tables. Please be more
careful about this if you would like to get a better and faster response.

The tables in cause are used for Sh interface notifications. To see
anything in them you probably would need to:
- have an AS that uses the Sh interface
- subscribe from that AS to certain events on the HSS
- modify data in the HSS
- wait for the Sh notification part to be fully implemented in the new
HSS branch.



-Dragos
