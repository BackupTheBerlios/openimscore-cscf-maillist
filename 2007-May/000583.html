Should be. If it is not, we will fix it.

For multiple I-CSCF you will need to do load balancing between them from the
DNS server for your home-domain (by default the open-ims.test)

For the S-CSCF you need to change the icscf database (see the
ser_ims/cfg/icscf.sql) and add all the S-CSCFs that you are going to start.
Then the S-CSCF could be balanced by the use of the generic S-CSCF
capabilities that you can assign in the database above for the S-CSCF and in
the HSS for the mandatory/optional capabilities required per user.
