Hello Tomasz,

Yes, you can install the ser_ims wherever you want, provided that you
change the configuration files accordingly.

About running as non-root, it should work. We had the task a while back
and I remember it fixed. However, there might still be some operations
that would require root permission.

For the libs installation dir, please look into the Makefiles for
ser_ims/lib. Then change the xcscf.sh to the new path.

No, there is no automatic way to change the ports in the config files as
well... it's not that complicated after all... Or you could use a sed
script to do this...

Cheers,
-Dragos
