Hi all,

Thanks to Erling, we have now 3rd party REGISTER. To use it you will
need to update to revision 79 and use the new scscf.cfg. Then you will
need to configure a Filter Criteria for matching the REGISTER message
and sending the 3rd party REGISTER to the corresponding Application Server.

Again, many thanks to Erling for taking the time and contributing this code.
