Hi Franz,

This should be more of a configuration issue. In the provided example,
only the Via was encrypted. But the exported functions should work also
for Route, Record-Route and Service-Route (also for other headers too,
if you would need that). It's just that the example configuration file
is a small one. The whole THIG is not 100% complete and some features
might not work. It was implemented by a student of mine and
unfortunately he kind of ran out of time on his diploma thesis. If you
spot any bugs, lacks, etc please post a bug/feature request so that we
could keep track of it.

-Dragos
