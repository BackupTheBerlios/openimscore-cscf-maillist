Hi Moshe,
actually, the open-source java project SIP-Communicator v1.0 (www.sip-communicator.org) implements the SIP protocol as a Java OSGi bundle.
To make it IMS compliant, you could get some of the code made available by us with the IMS-Communicator project (http://imscommunicator.berlios.de) - which is based on a older version of the SIP-Communicator.
